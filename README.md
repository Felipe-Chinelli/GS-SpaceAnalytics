## Descrição
O **Space Predictive Analytics** é uma plataforma inteligente e intuitiva de telemetria aeroespacial desenvolvida com React Native e Expo. Criado especificamente para o cenário de manutenção preditiva aeroespacial, o aplicativo atua de forma preventiva no monitoramento contínuo de sistemas de bordo essenciais — energia, temperatura, qualidade de link e órbita — processando anomalias térmicas e oscilações de sinal no exato momento de suas ocorrências. O diferencial tecnológico da nossa solução baseia-se na entrega de indicadores gráficos de alta performance projetados em tempo real matematicamente calibrados através de SVGs fluidos e um sistema de alerta preditivo avançado parametrizável por formulários de validação rigorosa persistidos localmente.

## Funcionalidades

- [x] Dashboard com indicadores em tempo real (simulado)
- [x] Sistema de alertas automáticos por limiar crítico inteligente
- [x] Persistência local robusta de limites com AsyncStorage
- [x] Navegação responsiva com abas virtuais nativas
- [x] Context Global para centralização das coletas e limiares operacionais
- [x] Formulário de configuração de limiares técnicos com validação de formato
- [ ] Integração nativa com NASA Open API (bônus planejado)

## Tecnologias

- **React Native** + **Expo Framework (SDK 51)**
- **AsyncStorage** (Gerenciamento durável de dados chave-valor)
- **React Native SVG** (Engine de desenhos matemáticos, gráficos lineares e radiais responsivos)
- **Lucide React Native** (Conjunto de ícones vetoriais modernos e minimalistas)
- **Tailwind CSS Style Engine** (Visual moderno, paleta de cores balanceada e tipografia de alta visibilidade)
- **Context API** (Controle centralizado do estado da missão em tempo real)

## Como Executar

### Pré-requisitos
- **Node.js** instalado (versão recomendada LTS)
- **Expo Go** instalado no celular físico (disponível na App Store ou Google Play)

### Instalação

1. Clone o repositório da aplicação:
   ```bash
   git clone https://github.com/seu-usuario/seu-repo.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd seu-repo/expo-space-app
   ```

3. Instale as dependências declaradas no package.json:
   ```bash
   npm install
   ```

4. Dispare o servidor de desenvolvimento Expo:
   ```bash
   npx expo start
   ```

5. Abra o aplicativo **Expo Go** em seu celular dispositivo físico e faça leitura do código QR em tela, ou rode em simuladores Android/iOS vinculados.
